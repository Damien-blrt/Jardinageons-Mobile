## 📚 Sommaire du projet

Ce dépôt contient la **partie mobile** du projet *Jardinageons*.  
L’application permet la gestion du potager directement depuis un smartphone (suivi des plantations, arrosages, récoltes, etc.).  
Toute la documentation et le suivi du développement sont disponibles sur le **Wiki** du projet.
